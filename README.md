## 特性

- **响应式设计** — 手机、平板、桌面端自适应
- **深色模式** — 支持亮色/深色主题切换
- **纯静态 HTML** — 零依赖，单文件包含全部样式与内容
- **中文排版** — 使用 Noto Serif SC / Noto Sans SC 字体

## 本地预览

```bash
# Python
python3 -m http.server 8080 -d h5

# Node.js
npx serve h5
```

直接在浏览器中打开 `h5/` 目录下的任意 HTML 文件也可以。

## 技术栈

纯 HTML + 内联 CSS + 内联 JavaScript，无任何构建工具或第三方依赖。
